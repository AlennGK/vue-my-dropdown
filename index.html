Guide

Requisites

  • jQuery >= 2.0
  • Vue >= 1.0

Install & Usage

You can add my-dropdown.min.js file, directly in the html code of your project. After you add the mixin in your Vue instance. 


new Vue({
	el: 'body',
	mixins: [window.myDropdown.mixin]
});

Or you can install the library using npm. 

$ npm install vue-my-dropdown

var myDropdown = require('vue-my-dropdown');

new Vue({
	el: 'body',
	mixins: [myDropdown.mixin]
});

Variable properties

myDropdown.mixin

This is a mixin object for Vue. Contains the vue component and the methods that you need for show and hide the dropdown dialog. In es6 this object is exported as default.

myDropdwon.component

It is the dropdown component for Vue.

myDropdown.version

String with the version number.

Component properties.

visible

Set the dropdown dialog visibility.

  • type:Boolean
  • required:false
  • twoWay:false
  • defaultfalse

id

Dialog unique id. It is necessary for the methods. See methods

  • type:String
  • required:true
  • twoWay:false

ref-id

Id of the html element where the dropdown dialog will show.

  • typeString
  • requiredtrue
  • twoWayfalse

position

Array with 4 elements indicating the dropdown dialog position with regard the html element. The possible values are: 

[left|center|right, top|center|bottom, left|center|right, top|center|bottom]

The 2 first array elements indicate the corner of html element. The 2 last array elements indicate the dropdown dialog corner. See example.

  • typearray
  • requiredfalse
  • twoWayfalse
  • default['right', 'top', 'left', 'top']

click-out

Flag indicate if the dropdown dialog will hide when click out of it.

  • typeBoolean
  • requiredfalse
  • twoWayfalse
  • defaultfalse
    • com

animation

Dialog animation when it show or hide. You can use the animation already created (See example) or create new animations See vue transitions for more info

  • typeString
  • requiredfalse
  • twoWayfalse
  • defaultmy-ani-scale

fixed

Flag indicating if the dropdown has an fixed css position or not. This property is useful if the ref html element has a fixed css position.

  • typeBoolean
  • requiredfalse
  • twoWayfalse
  • defaultfalse

Mixin methods

These methods are contains in the mixin variable.

showDropdown(id: String): void

Show the id dropdown.

hideDropdown(id: String): void

Hide the id dropdown.

isVisibleDropdown(id: String): Boolean

Return the id dropdown visibility.

Component events

dropdown::show(id: String)

You can use this event to show the dropdown dialog.

dropdown::hide(id: String)

You can use this event to hide the dropdown dialog.

dropdown::setVisibility(id: String, visibility: Boolean)

You can catch this event to know the dropdown dialog visibility.
